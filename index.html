Day One - June 19, 2015

First things first: All Hail to Torben Hoffman! Torben is pioneering the use of Elixir (and concurrency-oriented programming) to control Mindstorm robots. His talk is what got me fired up. Torben generously offered to help as I stumble along. (Thank you!)

I ordered Lego Mindstorm EV3 kit and a microSD card. They arrived yesterday.

As soon as I can pry it from the hands of my 12 year-old son Will's, I will follow Torben's instructions from his Elixir Conf 2015 slides and:

  1. Install the ev3dev Linux kernel with EV3 hardware drivers
  2. Install Erlang and Elixir on the card
  3. Install Torben's Elixir EV3 and EV3 Bluetooth libraries
  4. Boot the EV3 brick from the SD card.
  5. Build a robot (with Will of course)
  6. Write "hello robot" in Elixir and run it

Let's see how easy or hard it will be for me to get there.
